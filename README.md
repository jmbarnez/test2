# Novus
Novus is a top-down space action prototype built with LÖVE 11.5. Guide your starter ship through a physics-driven sector filled with asteroids, roaming enemies, and a central hub station while the ECS systems handle spawning, combat, and UI.

## Getting Started
1. Install LÖVE 11.5 from [love2d.org](https://love2d.org/).
2. Download or clone this repository.
3. Run `love .` (or `run.bat` on Windows) from the project folder.

## Contributing
